# 폰트 카탈로그

프로젝트 `fonts/` 디렉토리에 설치된 1,072개 폰트의 광고 소재용 분류.
크리에이티브 디렉터가 브리프 작성 시 이 카탈로그를 참조하여 소재에 적합한 폰트를 선택한다.

## 컴포지터에서 폰트 지정 방법

### 방법 1: 키워드 (Pretendard 전용)
```json
{ "font": "extrabold" }
```

### 방법 2: 파일명 (모든 폰트)
```json
{ "font": "GmarketSansTTFBold.ttf" }
{ "font": "NanumSquareRoundEB.ttf" }
{ "font": "Jalnan2TTF.ttf" }
{ "font": "SCDream7.otf" }
```

### 새 폰트 추가 시
1. TTF/OTF 파일을 `fonts/` 디렉토리에 복사
2. 이 catalog.md의 적절한 카테고리에 추가
