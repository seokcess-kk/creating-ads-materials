# Ad Copywriting

크리에이티브 브리프를 기반으로 채널별 광고 카피를 작성한다.

## 워크플로우

### 1. 브리프 분석
`_workspace/01_creative_brief.md`를 읽고 다음을 파악한다:
- 핵심 메시지와 톤앤매너
- 타겟 페르소나의 언어 습관
- 채널별 규격과 글자수 제한
- 카피 방향 지시사항

### 2. 앵글 매트릭스 설계
각 채널에 대해 사용할 앵글 조합을 결정한다:

| 앵글 | SNS | 검색 | 배너 |
|------|-----|------|------|
| 혜택 중심 | O | O | O |
| 문제 해결 | O | O | - |
| 사회적 증거 | O | - | O |
| 긴급성/희소성 | O | O | O |
| 감성 호소 | O | - | - |
| 호기심 유발 | O | - | O |

### 3. 채널별 카피 작성

#### SNS 카피 (Instagram, Facebook, TikTok)
- 헤드라인: 스크롤 스토핑 효과를 노린다. 질문, 숫자, 반전 활용
- 본문: 공감 → 솔루션 → 증거 → CTA 구조
- CTA: "지금 확인하기", "무료로 시작하기" 등 구체적 행동 동사
- 해시태그: 3-5개 관련 해시태그 제안

#### 검색 광고 카피 (Google, 네이버)
- 헤드라인: 검색 의도에 정확히 매칭. 핵심 키워드 자연스럽게 포함
- 설명문: 차별점 + 신뢰 요소 + CTA
- 반응형 검색 광고(RSA) 형식: 헤드라인 3개 + 설명 2개 세트

#### 디스플레이 배너 카피
- 극도로 간결: 핵심 메시지 1줄 + CTA
- 시각적 계층: 큰 텍스트(핵심) → 작은 텍스트(보조) → 버튼(CTA)
- 배너 사이즈별 카피 길이 조정

### 4. 변형 생성
- 채널별 최소 5개, 최대 10개 변형 생성
- 각 변형은 다른 앵글 또는 다른 어조를 사용한다
- 변형별 추천 활용 상황을 간단히 메모한다

### 5. 셀프 체크
- 글자수 제한 준수 여부
- 오탈자/문법 확인
- 톤앤매너 일관성
- CTA 명확성
- 민감한 표현/과장 광고 요소 제거
