# Ad Performance Review

광고 소재를 2단계로 평가하고 개선 방향을 제시한다.

## 전체 흐름

```
카피 + 이미지 완성
       ↓
[1차 리뷰] 개별 평가 + 경쟁사 분석 → 피드백
       ↓ (피드백 반영)
컴포지터 합성 → 최종 소재
       ↓
[2차 리뷰] 합성 소재 평가 → 업로드 판정
```

---

## 1차 리뷰: 카피 + 이미지 개별 평가

### 입력
- `_workspace/01_creative_brief.md` — 원래 의도 파악
- `_workspace/02_copywriter_copies.md` — 카피 평가 대상
- `_workspace/03_visual_designer_guide.md`, `_workspace/images/` — 이미지 평가 대상

### 워크플로우

#### 1. 경쟁사 분석
- WebSearch로 해당 업종/제품의 광고 사례 검색
- 경쟁사 광고의 메시지 전략, 비주얼 스타일, 차별화 포인트 분석
- 우리 소재 대비 강점/약점 비교

#### 2. 카피 평가 (1-5점)
| 기준 | 평가 요소 | 가중치 |
|------|----------|--------|
| 주목도 | 스크롤 멈춤 효과, 첫 문장 임팩트 | 25% |
| 명확성 | 핵심 메시지 전달력 | 25% |
| 설득력 | CTA 유도력, 행동 동기 | 20% |
| 채널 적합성 | 채널 톤, 글자수 준수 | 15% |
| 차별성 | 경쟁사 대비 독창성 | 15% |

#### 3. 이미지 평가 (1-5점)
| 기준 | 평가 요소 | 가중치 |
|------|----------|--------|
| 시각적 임팩트 | 피드 내 눈에 띄는 정도 | 25% |
| 메시지 전달 | 이미지만으로 의도 파악 | 20% |
| 브랜드 적합성 | 제품/서비스 이미지 일치 | 20% |
| 기술적 품질 | 해상도, 구도, 컬러 | 15% |
| 텍스트 공간 | 카피 오버레이 여백 확보 여부 | 20% |

#### 4. 카피-이미지 조합 추천
- 어떤 카피 변형과 어떤 이미지를 조합할지 추천
- 조합별 시너지 예측

#### 5. 합성 전 피드백
- 컴포지터에 전달되기 전에 수정해야 할 사항을 구체적으로 제시
- "더 좋게" 같은 추상적 조언 금지 → 실행 가능한 수정안 제시

## 2차 리뷰: 최종 합성 소재 평가

### 입력
- `_workspace/images/final/` — 합성된 최종 소재
- `_workspace/04_review_round1.md` — 1차 피드백 반영 여부 확인

### 워크플로우

#### 1. 합성 소재 평가 (1-5점)
| 기준 | 평가 요소 | 가중치 |
|------|----------|--------|
| 텍스트 가독성 | 배경 대비 텍스트 명확도, 폰트 크기 | 25% |
| 레이아웃 밸런스 | 텍스트-이미지 배치 균형, 시각적 계층 | 20% |
| 텍스트-이미지 조화 | 카피 톤과 비주얼 무드의 일체감 | 20% |
| CTA 명확성 | CTA 버튼/텍스트 시인성, 클릭 유도력 | 15% |
| 채널 최적화 | 해당 채널 피드에서의 노출 효과 | 10% |
| 브랜드 일관성 | 소재 간 통일감, 브랜드 요소 배치 | 10% |

#### 2. 1차 피드백 반영 확인
- 1차 리뷰에서 제시한 수정 사항이 반영되었는지 체크

#### 3. 소재 간 비교 순위
- 전체 소재를 종합 점수 기준으로 순위 매기기
- 각 소재의 최적 활용처 추천

#### 4. A/B 테스트 설계
- 테스트 우선순위가 높은 소재 조합 선별
- 가설, 측정 지표, 예상 소요 기간

#### 5. 최종 판정
- **업로드 가능** (4점 이상): 바로 광고 플랫폼에 사용
- **수정 후 재평가** (3-4점): 구체적 수정 후 재합성 필요
- **재제작** (3점 미만): 카피 또는 이미지부터 재작업
