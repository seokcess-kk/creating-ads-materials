# Ad Creative Brief

제품/서비스 정보와 캠페인 목표를 분석하여 광고 크리에이티브 브리프를 작성한다.

## 워크플로우

### 1. 입력 분석
사용자 요청에서 다음을 추출한다:
- 제품/서비스명, 특징, 가격대
- 캠페인 목표 (인지도, 트래픽, 전환, 앱 설치 등)
- 타겟 대상 (명시된 경우)
- 예산 규모 (명시된 경우)
- 경쟁사/레퍼런스 (명시된 경우)
- 원하는 채널 (명시되지 않으면 전체 채널)

누락된 핵심 정보가 있으면 합리적으로 추론하되, 추론한 부분을 브리프에 `[추정]` 태그로 표시한다.

### 2. 타겟 페르소나 정의
- 인구통계 (나이, 성별, 지역, 직업)
- 심리적 특성 (관심사, 가치관, 라이프스타일)
- 페인포인트 (해결하고 싶은 문제)
- 미디어 소비 패턴 (주로 사용하는 플랫폼, 시간대)
- 구매 여정 단계 (인지 → 고려 → 결정)

### 3. 콘셉트 도출
3가지 콘셉트 방향을 도출한다:
- 각 콘셉트에 핵심 메시지, 톤앤매너, 비주얼 무드를 포함
- 차별화 포인트를 명확히 제시
- 가장 적합한 1개를 추천하되 이유를 설명

### 4. 채널별 전략 수립
대상 채널별로:
- 소재 유형과 규격
- 핵심 메시지 방향 (채널 특성에 맞게 변형)
- 카피/비주얼 비중

### 5. 작업 지시 작성
카피라이터와 비주얼 디자이너에 대한 작업 지시:
- 작업 순서 판단: 콘셉트가 비주얼 중심이면 비주얼 우선, 메시지 중심이면 카피 우선
- 협업 방식: 상호 의존도가 높으면 병렬 + 피드백, 독립적이면 순차
